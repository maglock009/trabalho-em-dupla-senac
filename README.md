# Plataforma Educativa de Aprendizado Web – HTML, CSS e Boas Práticas 

## Introdução
### O que é
Esta plataforma é uma ferramenta educativa web para aprendizado prático de HTML, CSS, JavaScript e conceitos de front-end, com foco em boas práticas, semântica e estruturação de páginas.

O sistema permite que o usuário explore um sistema rudimentar similando o acesso ao acervo da biblioteca, possibilitando uma experiência de utilização do sistema da biblioteca. 


## Funcionalidades

### Módulos de Aprendizado
. HTML Básico: Estrutura, títulos, parágrafos, links, imagens<br>
. Listas e Tabelas: ul,ol,li,table,tr,td<br>
. Containers e Divs: div e span para organização de layout<br>
. Semântica: header, footer, nav, article, section<br>
. Mídia: video, audio, iframe<br>
. CSS Avançado: Seletores, pseudo-classes, box model, alinhamento de texto e cores<br>
. Formulários: Inputs básicos e avançados, submissão, validação de campos<br>
. Formulários e Interação<br>
. Submissão de dados de forma simulada ou real via MySQL<br>
. Validação de campos obrigatórios<br>
. Visualização de resultados imediatos<br>
. Registro opcional em banco de dados para análise de aprendizado<br>


## Pré-requisitos

. Python 3.10 ou superior<br>
. Flask<br>
. MySQL<br>
. Navegador Chrome, Firefox, Edge, etc.<br>
. Git<br>

## Como Executar

### Clonar o repositório:

git clone https://github.com/maglock009/trabalho-em-dupla-senac/<br>
cd exercicio_modulo2

### Instalar dependências:

pip install flask<br>
pip install mysql-connector-python<br>
pip install bcrypt<br>

## Executar a aplicação:

. python app.py <br>
.Acessar no navegador: http://localhost:5000<br>

### Backend
.Python 3.10+ – Linguagem principal do projeto<br>
